## 使用规则

- 复盘记录用于追溯事实、解释、建议、返工原因和改进输入。
- 执行任务时不应直接把复盘建议当成当前规则。
- 需要吸收为规则的内容，应同步更新到对应当前基线文档，并在复盘中记录确认结果。
